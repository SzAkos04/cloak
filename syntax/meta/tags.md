# Tags Reference

A list of tags used in the Syntax Vault with descriptions.

| Tag        | Description                                   |
| ---------- | --------------------------------------------- |
| `#syntax`  | Notes related to programming language syntax. |
| `#grammar` | Notes on grammar rules, parsing, and lexing.  |
| `#example` | Code snippets, sample usages, edge cases.     |
| `#meta`    | Vault metadata, configuration, and templates. |
| `#todo`    | Items or notes that require further work.     |

---

Use tags consistently to maintain organization and improve searchability.
